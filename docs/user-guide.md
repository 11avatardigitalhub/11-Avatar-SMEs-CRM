## 🚀 Getting Started

### Creating Your Account


1. Visit **[https://11avatardigitalhub.github.io/lead2revenue/](https://11avatardigitalhub.github.io/lead2revenue/)**

2. Click on **"Get Started Free"** or **"Register"**
3. Fill in your details:
   - Full Name
   - Email Address (this will be your login)
   - Phone Number
   - Company Name
   - Password (must meet security requirements)
4. Click **"Create Account"**
5. Verify your email address by clicking the link sent to your inbox
6. Complete your profile setup

### Logging In

1. Visit the login page
2. Enter your registered email and password
3. Optionally check **"Remember Me"** for 30-day persistent login
4. Click **"Login"**
5. If you have 2FA enabled, enter the code from your authenticator app

### Navigating the Interface

The application has a clean, intuitive interface:

- **Header:** Navigation bar with main menu items
- **Sidebar (Internal Pages):** Module navigation for quick access
- **Content Area:** Main working area displaying the active module
- **Footer:** Status information and quick links

### Setting Up Your Profile

1. Click on your avatar in the top-right corner
2. Select **"Profile"**
3. Update your:
   - Profile picture
   - Contact information
   - Notification preferences
   - Security settings (2FA, password change)

## 📋 Lead Management

### Creating a New Lead

1. Navigate to **Leads** from the sidebar
2. Click **"+ New Lead"** button
3. Fill in the lead form:

| Field | Required | Description |
|-------|----------|-------------|
| **Name** | Yes | Full name of the lead |
| **Email** | No | Email address |
| **Phone** | Yes (or Email) | Contact number |
| **Company** | No | Company name |
| **Source** | Yes | Where the lead came from |
| **Requirement** | No | What they're looking for |
| **Budget Range** | No | Estimated budget |
| **Timeline** | No | Expected decision timeline |
| **Notes** | No | Any additional information |

4. Assign the lead to a team member
5. Add tags for easy categorization
6. Click **"Save Lead"**

### Managing Leads

**Lead Statuses:**
| Status | Description |
|--------|-------------|
| **New** | Freshly created, not yet contacted |
| **Contacted** | Initial contact made |
| **Qualified** | Meets criteria, potential customer |
| **Proposal Sent** | Proposal/quotation shared |
| **Negotiation** | In discussion phase |
| **Won** | Successfully converted |
| **Lost** | Did not convert |
| **Disqualified** | Not a fit |

**Bulk Actions:**
- Select multiple leads using checkboxes
- Perform bulk operations:
  - Assign to team member
  - Change status
  - Add tags
  - Export to CSV/Excel
  - Delete

### Lead Scoring

Leads are automatically scored based on:
- Completeness of information
- Engagement level (email opens, clicks)
- Source quality
- Budget range
- Timeline urgency

Higher scores indicate hotter leads that need immediate attention.

### Importing Leads

1. Click **"Import"** button on the Leads page
2. Download the CSV template
3. Fill in your lead data following the template format
4. Upload the CSV file
5. Map columns to CRM fields
6. Review and confirm import

## 🧾 GST Invoicing

### Creating an Invoice

1. Navigate to **Invoices** from the sidebar
2. Click **"+ Create Invoice"**
3. Select the client
4. Choose invoice type:
   - **B2B:** Business to Business (with GSTIN)
   - **B2C:** Business to Consumer
   - **Export:** International clients
   - **SEZ:** Special Economic Zone

5. Add line items:

| Field | Description |
|-------|-------------|
| **Description** | Service/product description |
| **HSN/SAC Code** | GST classification code |
| **Quantity** | Number of units |
| **Rate** | Price per unit |
| **GST Rate** | 0%, 5%, 12%, 18%, or 28% |

6. Review auto-calculated taxes:
   - **Intra-state:** CGST + SGST (split equally)
   - **Inter-state:** IGST (full amount)

7. Set due date and payment terms
8. Add notes and terms/conditions
9. Click **"Generate Invoice"**

### E-Invoice Generation

For B2B invoices above ₹50,000:
1. After creating invoice, click **"Generate E-Invoice"**
2. System submits to IRP (Invoice Registration Portal)
3. IRN (Invoice Reference Number) is generated
4. QR code is embedded in the invoice
5. E-Invoice is ready for sharing

### E-Way Bill Generation

For transporting goods above ₹50,000:
1. Click **"Generate E-Way Bill"**
2. Enter transport details:
   - Transporter ID/Name
   - Vehicle Number
   - Approximate Distance
3. E-Way Bill number is generated
4. Valid for duration based on distance

### Invoice Actions

- **Send via Email:** Send invoice directly to client
- **Share via WhatsApp:** Send invoice link on WhatsApp
- **Download PDF:** Download formatted PDF copy
- **Print:** Print physical copy
- **Record Payment:** Mark payment received
- **Send Reminder:** Send payment reminder to client

### Payment Link

Each invoice includes a payment link that clients can use to pay online via:
- UPI (Google Pay, PhonePe, Paytm)
- Credit/Debit Cards
- Net Banking
- Bank Transfer (NEFT/RTGS/IMPS)

## 💰 Payment Collection

### Recording a Payment

1. Navigate to **Payments** from the sidebar
2. Click **"+ Record Payment"**
3. Select the invoice
4. Enter payment details:
   - Amount received
   - Payment method (UPI, Bank Transfer, Cheque, Cash)
   - Payment date
   - Transaction reference number
5. Click **"Record Payment"**

### Payment Reconciliation

1. Go to **Payments → Reconciliation**
2. Select date range
3. System matches:
   - Bank statements with recorded payments
   - Gateway transactions with invoices
4. Review matched and unmatched items
5. Resolve discrepancies

### Refund Processing

1. Find the payment in the payment list
2. Click **"Process Refund"**
3. Enter refund amount (full or partial)
4. Provide refund reason
5. Click **"Process"**
6. Refund is initiated through the original payment gateway

## ❓ Frequently Asked Questions

### General

**Q: Is there a free trial?**
A: Yes! All plans include a 14-day free trial. No credit card required.

**Q: Can I cancel anytime?**
A: Yes, you can cancel your subscription at any time. You'll retain access until the end of your billing period.

**Q: Is my data secure?**
A: Absolutely. We use AES-256 encryption, TLS 1.3, and comply with GDPR, DPDP Act 2023, and IT Act 2000.

### Invoicing

**Q: Does the system handle GST automatically?**
A: Yes! Simply enter HSN/SAC codes and select GST rates. CGST/SGST/IGST are calculated automatically based on intra/inter-state supply.

**Q: Can I generate E-Invoices?**
A: Yes, for B2B invoices above ₹50,000. IRN and QR codes are generated via IRP integration.

### Payments

**Q: What payment methods are supported?**
A: UPI, Credit/Debit Cards, Net Banking, Bank Transfer (NEFT/RTGS/IMPS), and digital wallets.

**Q: How long do payments take to reconcile?**
A: UPI payments are instant. Bank transfers typically reconcile within 1-2 business days.
