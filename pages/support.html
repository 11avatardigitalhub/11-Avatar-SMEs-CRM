Frequently Asked Questions

How do I create a GST-compliant invoice?

Navigate to Invoices → Click "Create Invoice" → Fill in client details with GSTIN → Add line items with HSN/SAC codes → Select appropriate GST rate (0%, 5%, 12%, 18%, 28%) → The system automatically calculates CGST/SGST or IGST based on inter/intra-state supply → Click "Save & Generate". For E-Invoicing (B2B >₹50,000), the IRN and QR code are auto-generated via the IRP portal.

How does the WhatsApp integration work?

Our WhatsApp integration connects via CloudWA (cloudwa.11avatardigitalhub.cloud) using the same Firebase project. Login to CloudWA using SSO → Scan QR code with WhatsApp on your phone → Once connected, you can send/receive messages, use templates, set up automation rules, and send bulk broadcasts. All messages are synced in real-time with the CRM.

What payment gateways are supported?

We support Razorpay (Indian market - UPI, cards, netbanking), Stripe (International), PayPal, UPI Direct, and Bank Transfer (NEFT/RTGS/IMPS). Payment webhooks automatically update invoice status. You can generate payment links or collect payments directly via the integrated checkout.

How do I set up automated backups?

Go to Settings → Backup & Restore → Enable Auto-Backup → Choose frequency (Daily/Weekly/Monthly) → Select backup type (Full/Incremental/Config) → Set retention period. Full backups run every Sunday at 2 AM IST. Backups are stored in Firebase Storage with encryption. You can also trigger manual backups anytime.

How does the referral program work?

Partners get unique referral links/codes → When a referred lead converts to a paying client, commission is auto-calculated → Partners have tiers (Bronze, Silver, Gold, Platinum) with increasing commission multipliers → Multi-level commissions (up to 5 levels deep) → Payouts processed via bank transfer, UPI, or PayPal. Track everything in Reports → Referral Analytics.

Is the platform DLT compliant for SMS?

Yes. Our SMS module is fully TRAI DLT compliant. All promotional SMS require pre-approved DLT templates with Entity ID and Header ID. Transactional SMS (OTP, alerts) are sent via high-priority routes. Sending hours are restricted to 9 AM - 9 PM for promotional messages as per regulations.
