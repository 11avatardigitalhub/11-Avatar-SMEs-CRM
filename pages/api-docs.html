API Documentation

Complete REST API reference for 11 Avatar Digital Hub CRM. Base URL: https://11avatar-api.11avatardigitalhub.workers.dev

Introduction

The 11 Avatar Digital Hub API provides programmatic access to all CRM features including leads, clients, deals, invoices, payments, tasks, projects, WhatsApp messaging, email, SMS, and more.

All API requests must be made over HTTPS. The API follows RESTful conventions and uses JSON for request and response bodies.

GET /api/health Health Check

Check API health status. No authentication required.

Response 200

{
  "status": "healthy",
  "version": "2.0.0",
  "timestamp": "2026-07-16T10:30:00.000Z",
  "uptime": 86400
}

Authentication

All API requests (except health check) require authentication via Bearer Token (JWT) or API Key.

Authentication Methods

Use Firebase ID Token for user-based access or API Key for server-to-server integration.

POST /api/auth/login User Login

Authenticate user and receive JWT token.

Request Body

{
  "email": "user@example.com",
  "password": "your-password"
}

Response 200

{
  "success": true,
  "token": "eyJhbGciOiJSUzI1NiIs...",
  "expiresIn": 3600,
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "role": "admin"
  }
}
POST /api/auth/api-key Generate API Key

Generate API key for server-to-server access. Requires admin role.

Request Body

{
  "name": "My Integration",
  "permissions": ["read:clients", "write:invoices"]
}

Response 201

{
  "success": true,
  "apiKey": "adk_live_8f7a9b3c...",
  "prefix": "adk_live",
  "expiresAt": "2027-07-16T10:30:00.000Z"
}

Clients API

GET /api/clients List Clients

Retrieve paginated list of clients with filtering and sorting.

Query Parameters

ParameterTypeDefaultDescription
pageinteger1Page number
limitinteger25Items per page (max 100)
statusstring-Filter by status
searchstring-Search name/email/phone
sortBystringcreatedAtSort field
sortOrderstringdescasc or desc

Response 200

{
  "success": true,
  "clients": [
    {
      "id": "uuid",
      "companyName": "Acme Corp",
      "email": "contact@acme.com",
      "phone": "9876543210",
      "status": "active",
      "createdAt": "2026-07-16T10:30:00.000Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 25,
    "total": 150,
    "totalPages": 6
  }
}

Error Codes

CodeDescription
200Success
201Created
400Bad Request - Invalid parameters
401Unauthorized - Invalid/missing token
403Forbidden - Insufficient permissions
404Not Found - Resource doesn't exist
429Too Many Requests - Rate limit exceeded
500Internal Server Error
503Service Unavailable

Error Response Format

{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": [
      {
        "field": "email",
        "message": "Invalid email format"
      }
    ]
  }
}
