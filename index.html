Frequently Asked Questions

Is it really free? +

Yes! Our Free Forever plan includes all core features: unlimited leads, pipeline management, client tracking, revenue monitoring, GST invoices, and WhatsApp integration for up to 5 team members. No credit card required. No time limit.

Is my data secure? +

Absolutely. We use Google Firebase infrastructure with bank-grade AES-256 encryption. Your data is stored in Indian servers (asia-south1, Mumbai). We are GDPR, DPDP Act 2023, and IT Act 2000 compliant.

Can I use it on mobile? +

Yes! 11 Avatar Digital Hub is a PWA (Progressive Web App). Install it on your Android or iOS device for a native app-like experience. Works offline too! No app store download needed.

How is WhatsApp integrated? +

We support the official WhatsApp Business API via CloudWA integration. Send messages, use pre-approved templates, manage chats, set up auto-replies, and send bulk broadcasts — all from within the CRM.

Can I import my existing data? +

Yes! Import your existing leads, clients, and contacts via CSV, Excel (.xlsx), or JSON files. We provide downloadable templates for each entity type with the correct column format.
