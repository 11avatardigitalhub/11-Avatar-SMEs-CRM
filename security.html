Enterprise-Grade Security

Your data is protected with industry-leading security practices. We're committed to keeping your business safe.

Infrastructure Security

Our platform runs on Google Cloud Platform (GCP) with data hosted in Mumbai, India (asia-south1 region), ensuring data residency compliance for Indian businesses.

Encryption at Rest

All data is encrypted using AES-256 encryption. Database files, backups, and attachments are all encrypted at rest by default.

Encryption in Transit

All communication uses TLS 1.3 with perfect forward secrecy. API endpoints enforce HTTPS with HSTS preloading.

Network Security

Firewall rules, DDoS protection via Google Cloud Armor, and IP whitelisting for enterprise customers.

Access Control

8-level RBAC with granular permissions. Multi-tenant data isolation ensures complete separation between organizations.

Session Management

Configurable session timeouts, concurrent session limits, and automatic logout on suspicious activity.

Backup & Recovery

Automated daily backups with 30-day retention. Weekly full backups stored for 12 weeks. Point-in-time recovery available.

Application Security

Two-Factor Authentication (2FA)

Support for TOTP authenticator apps, SMS, and email-based 2FA. Enforceable by organization policy.

API Key Authentication

Secure API access with scoped API keys. Each key has granular permissions and can be revoked instantly.

Input Validation & Sanitization

All user inputs are validated and sanitized. Protection against XSS, CSRF, SQL injection, and OWASP Top 10 vulnerabilities.

Audit Logging

Comprehensive audit trail for all data access, modifications, and administrative actions. Immutable logs retained for 1 year.
